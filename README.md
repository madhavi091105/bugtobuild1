# Backend Task – Contributors Data

This repository contains work done as part of an assigned backend task.

## What was done

- Took contributor data provided in JSON format
- Inserted the JSON data into a MySQL database using PHP
- Created a database connection file (`config.php`)
- Fetched data dynamically from the database using PHP
- Returned database data in JSON format for frontend use
- Removed dependency on static JSON files
- Made the website data fully dynamic (DB-driven)

## Tech Used

- PHP
- MySQL
- XAMPP

## Files

- `config.php` – database connection
- `insert_json.php` – insert JSON data into database
- `get_contributors.php` – fetch data from database as JSON

## Note

This work was done as part of an assigned task and not as an independent project.
